# Ewha-Recipe

<div align="center"><img src="https://i.ibb.co/DL2dr9y/Kakao-Talk-20240209-184644484.png" alt="ewha-recipe-logo" border="0" width="100%"></div>

  
## 1. 프로젝트 소개
Ewha Recipe는 요리를 사랑하는 사람들이 자신만의 레시피를 공유하고, 전 세계 다른 사람들의 레시피를 발견할 수 있는 플랫폼입니다.<br>
이 웹사이트는 사용자가 레시피를 쉽게 업로드하며, 다양한 요리법을 탐색할 수 있는 기능을 제공합니다.
